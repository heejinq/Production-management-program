# 생산관리 시스템<br>
spring legacy projent를 이용해서 생산관리 시스템을 만들었다.<br>
이틀만에 후다닥 만드느라 css는 없다 ㅎㅎ<br>
<br>

STS 3.9.15<br>
JDK 1.8<br>
Tomcat 9.0<br>
Spring 5.0.7<br>
Maven 2.5.1<br>
MySQL<br>
